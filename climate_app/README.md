# climate_app

App en Flutter para monstrar el clima y la ciudad actual en telefono y wereable

## Set up

1. 'posicionarte sobre el archivo main.dart dentro de la carpeta lib'
2. 'Presionar F5'

## Resultado

La app muestra el clima en conjunto con un icono de material design (nube), muestra los grados y la ciudad de ubicación sin mostrar otros datos sensibles
